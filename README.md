# AllTheTiers

A mod for desynced which aims to add further tiers to many entities.
Also unlocks higher base game tiers that are not yet accessible.

Currently implemented upgrades:
* Buildings:
	* Garage (8)
	* Storage (48)
	* 1x1 Building
	* 2x1 Building
	* 2x2 Building
	* 2x3 Building
* Bots:
	* Transporter
	* Twinbot
	* Rock
	* Hound
* Drones:
	* Transfer Drone
* Components:
	* Miner
	* Transporter
	* Smaller Wind Turbine
	* Assembler
